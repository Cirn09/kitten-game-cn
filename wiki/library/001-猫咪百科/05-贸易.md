  小猫与其他种族进行交易以获取<a href="?file=003-资源大全/005-资源介绍">资源</a>。在和一些种族进行交易时，获得的资源收益率会在一年中不同的季节发生变化。
  <br>派遣探险队可以找到新的种族，如果你达到了先决条件的要求，就能发现这个种族。派遣探险队花费1000喵力。
  <br>如果没有可以发现的贸易伙伴将会退还900喵力。每一笔成功的交易都有机会获得某些额外的资源。

## 种族态度

  <ul>
   <li>敌对的文明交易斑马和狮鹫，有一定几率得不到任何资源（“ <em>[种族]</em> 无缘无故恨你”）</li>
   <ul>
   <li>斑马初始30% 狮鹫初始 15%。</li>
   </ul>
   <li>友好的文明有机会给你15-25%的额外资源（“ <em>[种族]</em> 认为小猫是最可爱的”）。</li>
   <li>在工坊<a href="?file=001-猫咪百科/04-工坊/01-升级#商队驿站">商队驿站</a>后，每个交易站将敌对事件发生的概率平均降低0.35%，并且友好事件触发概率增加0.175%。<br>即 30% / 1.75% ≈ 86个交易站将会让斑马不再仇恨你（购买<a href="?file=001-猫咪百科/03-科学/02-玄学#外交手段">外交手段</a>后为56个）。</li>
  </ul>


## 额外资源
  <ul>
   <li>进行任何交易都有机会获得<a href="?file=003-资源大全/37-蓝图">蓝图</a>和<a href="?file=003-资源大全/44-香料">香料</a>。</li>
   <ul>
<li> 有10%的机会获得一个蓝图。</li>
<li> 有35%的机会获得至少25香料（数量会随贸易比率和大使馆数量的增加而增加）。</li>
<li> 和与斑马交易获取钛不同，获得蓝图的概率是固定的，不会受其他资源影响而增加。</li>
<li> 获取香料的概率可由<a href="#大使馆">大使馆</a>提升  </li>
   </ul>
  </ul>

 #### 利维坦献祭
 <ul>
<li>如果你已经建立了一个或多个<a href="?file=001-猫咪百科/06-宗教/001-庙塔#神印">神印</a>，并且至少有一个天角兽，你会慢慢地产生<a href="?file=003-资源大全/47-死灵兽">死灵兽</a>。死灵兽可以喂给利维坦来增加能量并获得额外的好处。</li>
<li>在贸易标签中，利维坦有一个“献祭死灵兽”按钮。单击此按钮的工作原理如下：</li>
<ul>
 <li> 如果你有死灵兽，上古神会很高兴，他们的能量将会增加1。</li>
 <li> 如果你没有死灵兽，上古神会生气，他们会立即离开，从贸易窗口消失。</li>
</ul>
</ul>

 #### 利维坦能量
<ul>
<li> 在游戏一开始或重置后，利维坦的能量均为0。每献祭利维坦一个死灵兽，能量增加1。</li>
<li> 能量有如下效果：</li>
<ul>
 <li> 每点能量延长利维坦交易前的停留时间1年，停留时间为(5+能量点数)天。</li>
 <li> 每点能量使利维坦在交易后停留更长的时间，停留时间为 1年+(能量点数×100)天。</li>
 <li> 每点能量可以获得+ 2%的交易资源增益。</li>
 <li> 从1.4.1.0开始，利维坦的能量上限为 5 + 5 × 神印数量。</li>
 <ul>
 <li> 利维坦能量上限还可被政策死灵统治和挑战黑暗天空的完成次数加成。(四舍五入取整)</li>
 </ul>
</ul>
</ul>

### 惹怒看不见的黑手
<ul>
<li> 在工坊<a href="?file=001-猫咪百科/04-工坊/01-升级#看不见的黑手">看不见的黑手</a>升级完后，利维坦贸易界面会出现“惹怒看不见的黑手”的按钮。</li>
<li> 点击“惹怒看不见的黑手”的按钮作用：</li>
 <ul>
 <li>消耗一定数量的时间水晶，则会使的黑币价格下降0.3 - 0.1 \* [0~1]之间的随即数</li>
 <li>当黑币价格低于550时，惹怒看不见的黑手将不可点击。</li>
 </ul>
<li> “惹怒看不见的黑手”的时间水晶价格是根据多方面有关具体公式为：</li>
 <ul>
 <li> 游戏内公式是以前写的且没再更新，所以政策方面增加难得素或者贸易效率都不会影响惹怒看不见黑手的价格。<br>(注意下面的加成都是默认，比如你的领袖特质不是商人也是会算商人加成的)</li>
 <li> 一年难得素产量都跟利维坦贸易获得水晶数量 N = 空间生产加成 \* 太空制造加成 \* 交易所的贸易加成 \* 商人领袖的贸易加成 \*<br> 利维坦能量加成 \* 游戏1年的总tick次数 \* 微型亚空间反应装置加成 \* 基础月球前哨每秒产生的难得素 \* 0.25 \* 0.98 / 5000 (利维坦贸易)</li>
 <li> 也就是只有月球前哨、太空电梯、轨道阵列、工厂、交易所的数量有关。（想降低价格只能出售，关闭是没有作用的）</li>
 <li> A = 1.13 (平均50周期难得素产量的系数) \* N / 4000， B = 2.4 (红月年的难得素系数) \* N / 4000， C = A + 1.13 \* （资源回复 \* 耀变体的加成）\* 时计炉系数</li>
 <li> D =（在A，B，C中取最大一个数 ）\* 8000466.693057134 \* ln(1100 / 当前黑币价格) </li>
 <li> 然后在 D 和 256 中取最大一个数为E。
 <li> x = lnE / ln2。（即x最小为8）
 <li> 时间水晶价格为2的x次方，x需要向上取整。</li>
</ul>
