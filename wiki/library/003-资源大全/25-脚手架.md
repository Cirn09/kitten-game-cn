# 脚手架
<p>
	<strong>
		脚手架
	</strong>
	是
	<a href="?file=003-资源大全/005-资源介绍#工艺制作资源">
		工艺制作资源
	</a>
	。
</p>

### 获取
<ul>
	<li>
		在
		<a href="?file=001-猫咪百科/04-工坊/02-工艺">
			工坊
		</a>
		制作，
		每个脚手架
		需要50
		<a href="?file=003-资源大全/24-木梁">
			木梁
		</a>
		来制作
		。工艺制作效率可以增加制作脚手架的数量。
	</li>
	<li>
		也可以与蜥蜴贸易获得，需设立一定数量的
		<a href="?file=001-猫咪百科/05-贸易#大使馆">
			大使馆
		</a>。
	</li>
</ul>
